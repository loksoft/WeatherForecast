# Weather Forecast App

## Overview
An advanced weather forecasting Android application built using Kotlin, Clean Architecture, MVVM, Hilt, Coroutines, and Jetpack components.

## Features
- Current weather information
- Hourly and weekly forecast
- Offline-first support

## Tech Stack
- Kotlin
- Android SDK
- MVVM + Clean Architecture
- Hilt Dependency Injection
- Retrofit
- Room Database
- Coroutines + Flow
